# Plataforma Judo - Ready to deploy

This is a ready-to-upload Next.js 14 TypeScript starter configured for Supabase auth (email+password). Replace environment variables in Vercel before deploying.

## How to use
1. Edit `./env.example` values with your Supabase project info.
2. Upload to GitHub and connect to Vercel.
3. In Vercel, add the environment variables listed in `env.example` (do NOT add secret keys to the repo).
4. Redeploy (clear cache if needed).
